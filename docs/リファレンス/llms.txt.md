# llms.txt

このリファレンスサイトでは、[llmstxt.org](https://llmstxt.org/) の標準に従った `llms.txt` ファイルを提供しています。

## llms.txt とは

llms.txt は、AI やLLM（Large Language Model）がプロジェクトやライブラリを効率的に理解できるよう設計されたファイル形式です。プロジェクトの重要な情報を構造化されたテキスト形式で提供し、AIアシスタントがより正確で有用なサポートを提供できるようにします。

## 提供ファイル

GASsma リファレンスでは、以下の2つのファイルを提供しています：

### 📄 llms.txt（目次版）
- **アクセス URL**: [/llms.txt](/llms.txt)
- **内容**: llmstxt.org 標準に沿ったキュレーション済みの目次。GASsma の概要と、各リファレンスページへのリンク・説明のリスト
- **対象**: AI がトピックごとに必要なドキュメントページを見つけて参照する場合

### 📄 llms-full.txt（完全版）
- **アクセス URL**: [/llms-full.txt](/llms-full.txt)
- **内容**: GASsma の完全な技術仕様、エッジケース、実装詳細を 1 ファイルにまとめたもの
- **対象**: 詳細な実装や高度な使用法、トラブルシューティングが必要な場合や、コンテキストに全文を渡したい場合

## 使用方法

これらのファイルは主に以下の用途で使用されます：

1. **AI アシスタントとの連携**: Claude や ChatGPT などの AI ツールにファイル内容を提供することで、より正確な GASsma に関するサポートを受けられます

2. **プロジェクト理解の促進**: 新しい開発者がプロジェクトを理解する際の参考資料として

3. **ドキュメント検索**: 特定の機能や API について素早く情報を見つけたい場合

## ファイル形式の特徴

- **マークダウン形式**: 人間にも機械にも読みやすい形式
- **標準準拠の目次**: llms.txt は H1 タイトル・要約・セクションごとのリンクリストという llmstxt.org の標準構造に準拠
- **実用的なコード例**: llms-full.txt にはすぐに使える TypeScript/JavaScript のサンプルコードを収録
- **包括的なリファレンス**: API の詳細から使用例まで網羅

## 更新情報

llms.txt ファイルは GASsma ライブラリの更新に合わせて定期的に更新されます。最新の情報については、常に上記の URL からアクセスしてください。
